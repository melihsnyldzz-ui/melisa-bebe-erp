# OpenClaw / Telegram Onay Notu

Bu proje şu anda local güvenli agent-loop olarak çalışır.

İleride OpenClaw veya Telegram benzeri bir kanal yalnızca STOP durumlarında insan onayı almak için değerlendirilebilir. Git push, deploy, canlı DB, hosting paneli, ödeme veya production ayarı gibi işlemler ayrı ve açık bir onay kanalına bağlanmadan otomatik yapılmamalıdır.
