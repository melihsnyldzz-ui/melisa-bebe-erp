# Kapsam Dışı: Telegram Onay Akışı

Bu proje artık Telegram onay akışı kullanmıyor.

Telegram, webhook, remote approval, gateway ve diger dis servis entegrasyonlari aktif sistemin parcasi degildir.

Bu dosyada aktif kurulum, bot, webhook veya dis onay akisi adimi tutulmaz. Eski Telegram akisi kullanilmayacak.

Lokal agent-loop icin kullanilacak komut:

- `npm run agent:loop:dry`
