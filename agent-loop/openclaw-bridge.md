# Kapsam Dışı: OpenClaw Entegrasyonu

Bu proje artık OpenClaw kullanmıyor.

OpenClaw, webhook, remote approval, gateway ve diger dis servis entegrasyonlari aktif sistemin parcasi degildir.

Bu dosyada aktif kurulum veya calistirma adimi tutulmaz. Eski OpenClaw komutlari kullanilmayacak.

Lokal agent-loop icin kullanilacak komut:

- `npm run agent:loop:dry`
